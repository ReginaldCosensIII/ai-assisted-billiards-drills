# ai-assisted-billiards-drills

> **Status: Prototype Complete — Core projection, calibration, and database persistence are implemented.**

This repository is the initial scaffold for a desktop-first billiards drill projection platform with future AI-assisted coaching capabilities.

## What This Repository Is

A documentation-first, agentic-workflow-ready foundation. All structural decisions are intentionally deferred until the scoping and architecture phases are complete.

## What This Repository Is Not

This is **not** a place for:
- Product feature implementation
- Final framework or technology choices
- Speculative architecture diagrams or code

## Top-Level Structure

| Directory   | Purpose                                              |
|-------------|------------------------------------------------------|
| `app/`      | Front-end / desktop application layer                |
| `backend/`  | Server-side services and APIs                        |
| `shared/`   | Code and types shared across app and backend         |
| `docs/`     | All project documentation                            |
| `scripts/`  | Developer and CI/CD automation scripts               |
| `tests/`    | Integration and end-to-end test suites               |
| `tooling/`  | Build tools, linters, and configuration utilities    |
