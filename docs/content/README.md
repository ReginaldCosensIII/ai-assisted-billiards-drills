# docs/content/

This folder holds user-facing content assets: copy, labels, instructional text, and media references for the billiards drill platform.

## Intended Use

- Drill names, descriptions, and instructional copy
- UI label strings and microcopy
- Reference images, diagrams, or media assets (or links to external storage)
- Localization files (if multilingual support is planned)

## Current Status

**Empty — Foundation stage.** Content work begins after the scope phase defines what user-facing content the product requires.
