# docs/architecture/

This folder contains system design documents and technology decision records for the platform.

## Intended Use

- High-level system architecture diagrams and narratives
- Technology selection rationale (frameworks, languages, databases)
- Data flow and component interaction models
- Security and deployment architecture

> **Note:** Individual technology choices should also be captured as ADRs in `docs/adrs/`. This folder is for broader architecture narratives.

## Current Status

**Empty — Foundation stage.** No architecture decisions will be documented here until the scope phase is complete.
