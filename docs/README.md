# docs/

This directory contains all project documentation, organized by phase and concern.

## Structure

| Folder          | Purpose                                                       |
|-----------------|---------------------------------------------------------------|
| `master-brief/` | Project vision, goals, and stakeholder context                |
| `scope/`        | Feature scope definitions and boundary decisions              |
| `architecture/` | System design documents and technology decisions              |
| `adrs/`         | Architecture Decision Records (ADRs)                          |
| `testing/`      | Testing strategy and quality assurance planning               |
| `workflow/`     | Agentic development workflow definitions and process guides   |
| `content/`      | User-facing content, copy, and media assets                   |
| `reports/`      | Progress reports and retrospectives                           |
| `prompts/`      | Curated prompts used in the agentic development workflow      |

## Current Status

**Foundation stage only.** All subdirectories are scaffolded. Documentation content will be added in subsequent phases.
